<br/></br>### 课程介绍:
深度学习是一门综合性的学科，需要用到大量的数学和计算机专业的知识，而大量的数学公式和数学推导往往让初学者望而却步，知难而退。在这门课程中，我们希望能够通过尽量少的数学和公式，在讲课过程中，没有使用传统的读ppt的方式，而是通过手写板书的方式，以及实际的编程实践，让大家能够入门深度学习这门课程，培养起大家对深度学习的兴趣，并能解决一些实际的问题。
